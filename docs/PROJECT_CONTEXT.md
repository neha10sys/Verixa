# VERIXA PROJECT CONTEXT

> Project Name: Verixa

Tagline:

> Proof of Skills, Not Just Claims.

------------------------------------------------------------------------

# PROJECT GOAL

Verixa is a MERN Stack platform where developers can prove their
technical skills instead of only claiming them.

The platform allows:

-   Developers to build verified profiles
-   Recruiters to find trusted developers
-   Skill Assessments
-   GitHub Verification
-   Resume Analysis
-   Trust Score
-   Project Verification
-   Certificates
-   AI based features
